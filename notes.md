# Interaktioner med en Bankomat

## Bankomaten har 11000 kr. På kontot finns 9000 kr.

- [x] Sätt in ett kort i bankomaten. (Bankomaten ska veta att ett kort är inne)
- [x] Mata in en felaktig pinkod (1234) i bankomaten. (Pinkoden ska vara sparad på kortet, men det är bankomaten som ska veta om rätt eller fel kod matats in)
- [x] Mata in korrekt pinkod (0123).
- [x] Ange 5000 kr att ta ut via bankomaten. Balansen ska tas från kontot som är kopplat till kortet.
- [x] Mata ut kortet ur bankomaten.
- [x] Mata in kortet igen, slå in pinkoden igen.
- [x] Ange 7000 att ta ut. (Nu ska det inte finnas pengar så det räcker på bankomaten)
- [x] Ange 6000 att ta ut. (Nu räckte bankomatens pengar precis, men inte pengarna på kontot)
- [x] Mata ut kortet ur bankomaten.
